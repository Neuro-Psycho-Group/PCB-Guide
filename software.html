EDA Tools

Electronic Design Automation software is where all PCB design begins. Here's what's available, which to choose, and how to start.

What is EDA?

What EDA Software Does

EDA (Electronic Design Automation) software combines two main tools in one package: a schematic editor and a PCB layout editor. They work together, with the schematic's netlist driving the layout.

The two editors are linked — changes in the schematic update the netlist, which flows into the PCB editor as ratsnest lines to route

Tool 1

Schematic Editor

Draw your circuit using symbols. Add nets, power flags, component values. Generates the netlist — the list of all connections.

Tool 2

PCB Layout Editor

Place footprints on a board. Route copper traces between pads. Set up copper fills, silkscreen, and board outline.

Output

Gerber Files

Manufacturing files exported from the layout editor. Sent to the fab house. One file per layer of the board.

Both tools share a component library — a database of symbols (for the schematic) and footprints (for the layout). Matching the right footprint to the right symbol is one of the most important and error-prone steps in PCB design.

In depth

The Main Tools

KI
KiCad
Free · Open source · kicad.org

The industry-standard open-source EDA tool. KiCad 7 and 8 are genuinely professional-grade tools used from bedroom projects to aerospace hardware. It has a steeper initial learning curve than EasyEDA, but the skills you build here transfer directly to professional environments.

Strengths
  • Completely free, forever
  • Excellent schematic editor
  • Push-and-shove router
  • 3D board preview
  • Massive library + community
  • Works fully offline
Weaknesses
  • First-time setup is confusing
  • Library management complex
  • Autorouter is basic
⭐ Best all-round choice. Learn KiCad and you've learned PCB design properly.
EE
EasyEDA
Free · Browser-based · easyeda.com

Browser-based EDA tool made by JLCPCB's parent company LCSC. Tightly integrated with JLCPCB ordering and LCSC component database. Very low friction to get started — no installation, huge library of components with available stock, one-click ordering.

Strengths
  • No installation required
  • Massive real-stock LCSC library
  • Direct JLCPCB ordering
  • Very easy for first board
  • LCSC BOM integration
Weaknesses
  • Requires internet connection
  • Less powerful schematic tools
  • Cloud-locked designs
Best for your very first board if you don't want to install software. Move to KiCad as you grow.
AD
Altium Designer
~£7,000/year · altium.com

The gold standard of professional EDA software. Used by the majority of hardware companies for production design. Exceptional schematic tools, advanced routing features (differential pairs, length matching, impedance control), and the best library management available. You will encounter it if you work in hardware engineering.

Strengths
  • Most powerful routing engine
  • Best constraint management
  • Industry standard professionally
  • Excellent simulation tools
Weaknesses
  • Extremely expensive
  • Not worth learning until paid to use it
  • Very complex for beginners
Don't buy or learn this until you're employed somewhere that uses it. KiCad produces professional results at zero cost.
Fx
Flux.ai
Free tier available · flux.ai

A newer browser-based EDA tool with AI-assisted features. Pitched at rapid prototyping and collaboration. The AI can suggest component connections and help with routing decisions. Interesting for exploration but still maturing as a product. Entirely cloud-based.

Strengths
  • AI-assisted design features
  • Good collaboration tools
  • Modern UI
Weaknesses
  • Cloud-only, no offline use
  • Smaller community
  • Less proven in production
Worth watching. Not yet a primary recommendation — use KiCad or EasyEDA until it matures further.

Getting started with KiCad

KiCad: First Steps

KiCad can feel overwhelming when you open it. Here's what the main components are and how they connect.

Download

Get the latest stable release from kicad.org/download. KiCad 8 is the current version. Install includes all libraries — total install is about 5GB. Worth every byte.

The KiCad workflow

01

Project Manager

The main KiCad window. Create a new project here (File → New Project). Every design is a project — it holds your schematic file (.kicad_sch), PCB file (.kicad_pcb), and library symlinks. Open each sub-tool from this window.

02

Schematic Editor (Eeschema)

Open this first. Draw your circuit using symbols from the symbol library (A to add a symbol). Wire components together (W). Add power symbols (P). When done, run ERC (Inspect → Electrical Rules Checker) to catch errors before layout.

03

PCB Editor (PCBnew)

Open after the schematic is complete. Import the schematic's netlist (Tools → Update PCB from Schematic). All your components appear as footprints connected by ratsnest lines. Place them, draw the board outline on Edge.Cuts layer, then route.

04

Plot / Generate Gerbers

In PCBnew: File → Plot. Select all required layers. Set output format to Gerber. Also run File → Fabrication Outputs → Drill Files. Compress everything into a .zip and upload to your fab house.

Further learning

Learning Resources

YouTube

Phil's Lab

The best KiCad tutorial series online. Phil covers complete designs from schematic to ordering. Professional-grade content that's beginner-accessible. Essential watching.

YouTube

EEVblog

Dave Jones' channel covers PCB design, test equipment, and electronics fundamentals in depth. Particularly good for design-for-test and practical engineering tips.

Book

The Art of Electronics

Horowitz & Hill. The definitive electronics engineering reference. Not a PCB book specifically, but understanding the circuits you're laying out transforms your design quality.

Community

r/PrintedCircuitBoard

Post your layout for review before ordering — the community is generous with specific, actionable feedback. A genuine shortcut to avoiding mistakes that would take weeks to discover yourself.

Community

KiCad Forum

forum.kicad.info — the official KiCad community. Friendly, knowledgeable, and highly responsive to beginner questions.

Reference

IPC-2221 Standard

Generic standard for printed board design. Freely available in summarised form online. Defines trace width vs current, clearances, and via sizing formulae. The professional reference.
